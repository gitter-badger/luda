Button

Buttons can be created easly by Luda CSS calsses in various styles, sizes, actions and more.

Button categories

Button elements includes Solid Button, Hollow Button, Text Button, Checkbox Button, Radio Button and File Button.

# Solid buttons

Solid button has solid background, add classes btn btn-{$color} to a button to create a solid button.


<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-dark">Dark</button>
<button class="btn btn-light">Light</button>
<button class="btn btn-danger">Danger</button>

# Hollow buttons

Hollow button has hollow background, add classes btn btn-hollow-{$color} to a button to create a solid button.


<button class="btn btn-hollow-primary">Primary</button>
<button class="btn btn-hollow-secondary">Secondary</button>
<button class="btn btn-hollow-dark">Dark</button>
<button class="btn btn-hollow-light">Light</button>
<button class="btn btn-hollow-danger">Danger</button>

# Text buttons

Text button has solid background, add classes btn btn-{$color} to a button to create a solid button.


<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-dark">Dark</button>
<button class="btn btn-light">Light</button>
<button class="btn btn-danger">Danger</button>

Button tags

Button classes can be added to other html tags not only to button tag.

# Use other html tags as buttons

Just add .btn-* classes to html tag.

Link 

<a href="#" class="btn btn-primary">Link</a>
<input type="submit" value="Submit" class="btn btn-hollow-primary">
<input type="reset" value="Reset" class="btn btn-primary">
<input type="button" value="Button" class="btn btn-hollow-primary">

# Checkbox button

Add class .btn-check and button categories classes to a div tag with a input tag wrapped in.


<div class="btn-check btn-hollow-primary">
  <input type="checkbox" name="favriout_films" value="kill_bill">
  <label>Kill Bill</label>
</div>
<div class="btn-check btn-hollow-primary">
  <input type="checkbox" name="favriout_films" value="django">
  <label>Django</label>
</div>

# Radio button

Add class .btn-radio and button categories classes to a div tag with a input tag wrapped in.


<div class="btn-radio btn-hollow-primary">
  <input type="radio" name="decision" value="yes">
  <label>Yes, I agree.</label>
</div>
<div class="btn-radio btn-hollow-danger">
  <input type="radio" name="decision" value="no">
  <label>No, I disagree.</label>
</div>

# File button

Add class .btn-file and button categories classes to a div tag with a input tag wrapped in.


<div class="btn-file btn-primary">
  <input type="file">
  <label>Upload a file</label>
</div>
<div class="btn-file btn-hollow-primary">
  <input multiple type="file">
  <label>Upload files</label>
</div>

Button states

Button state can be changed to disabled, checked or actived.

# Disabled button

Add attribute disabled to a button to disabled it, if it's a checkbox button, radio button or file button, add attribute disabled to the wrapped input too.


<button disabled class="btn btn-primary">Disabled</button>
<div class="btn-file btn-hollow-primary" disabled>
  <input disabled type="file">
  <label>Disabled file</label>
</div>
<div class="btn-check btn-hollow-primary" disabled>
  <input disabled type="checkbox">
  <label>Disabled checkbox</label>
</div>

# Actived button

Add class .btn-active to a button to show actived state.


<button class="btn btn-hollow-primary btn-active">Actived</button>
<div class="btn-file btn-hollow-primary btn-active">
  <input type="file">
  <label>Actived file</label>
</div>
<div class="btn-radio btn-hollow-primary btn-active">
  <input type="radio" name="actived_radio">
  <label>Actived radio</label>
</div>

# Checked button

Add attribute checked to the wrapped in input of a checkbox button or radio button.


<div class="btn-radio btn-hollow-primary">
  <input checked type="radio" name="checked_radio">
  <label>Checked radio</label>
</div>
<div class="btn-check btn-hollow-primary">
  <input checked type="checkbox">
  <label>Checked checkbox</label>
</div>

Button modifiers

Button size can be changed with size modifiers.

# Small buttons Modifier

Add class .btn-small to a button element.


<button class="btn btn-primary btn-small">Small</button>
<button class="btn btn-hollow-primary btn-small">Small</button>
<button class="btn btn-text-primary btn-small">Small</button>
<div class="btn-file btn-hollow-primary btn-small">
  <input type="file">
  <label>Small file button</label>
</div>

# Large buttons Modifier

Add class .btn-large to a button element.


<button class="btn btn-primary btn-large">Large</button>
<button class="btn btn-hollow-primary btn-large">Large</button>
<button class="btn btn-text-primary btn-large">Large</button>
<div class="btn-check btn-hollow-primary btn-large">
  <input type="checkbox">
  <label>Large check button</label>
</div>

# Fluid buttons Modifier

Add class .btn-fluid to a button element.


<button class="btn btn-primary btn-small btn-fluid mb-small">Small fluid</button>
<button class="btn btn-hollow-primary btn-fluid mb-small">Fluid</button>
<div class="btn-file btn-primary btn-large btn-fluid">
  <input type="file">
  <label>Large fluid file button</label>
</div>
