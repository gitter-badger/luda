Attribute

Value

Description

data-toggle-target

Set a toggle target with a name.

data-toggle-for

Set an outside toggler for named toggle targets.

data-toggle

Set an inside toggler in a toggle target.

data-toggle-disabled

Disable toggle behaviour in an inside toggler.

Method

Params

Description

Target name | Target element

luda.toggle.deactive

Target name | Target element
