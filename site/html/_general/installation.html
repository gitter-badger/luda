Bundler is the recommand way to install luda, add luda gem to your Gemfile:

{% highlight ruby %} gem 'luda' {% endhighlight %}

then run bundle in command line.

{% highlight bash %} $ bundle {% endhighlight %}

You can also download luda through gem.

{% highlight bash %} $ gem install luda {% endhighlight %} {% include sample-title.html title="Step 2: Choose a Theme and Import it in application.sass or application.scss" %} {% highlight sass %} // Import your custom variable overrides. @import path_to_your_sass_variables // Import default theme for example. Don't include multiple themes toghther. @import luda/default {% endhighlight %}

If application.css exists in your project, rename it to .sass or .scss extension. And don't use require statement to import files, use @import instead.

{% include sample-title.html title="Step 3: Import JavaScript in application.js" %} {% highlight javascript %} //= require luda {% endhighlight %} {% include chapter-title.html title="Install in Other Ruby Apps" %}

If your project is based on Sprokets or Hanami, Luda assets will be added to load path automaticly, you just need require the assets in your project. Otherwise, you need manually register Luda assets, you can reference to Luda gem source code and your framework documentation to do that."
