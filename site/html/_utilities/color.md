## Usage

Color utility classes include `.c-primary`, `.c-secondary`, `.c-danger`,
`.c-dark`, `.c-light`, `.c-emphasis`, `.c-main`, `.c-muted`, `.c-inverse-emphasis`,
`.c-inverse-main` and `.c-inverse-muted`, see how they change text color.
